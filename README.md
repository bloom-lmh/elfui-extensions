# ElfUI Extensions

`extensions/` 存放非核心、可选安装的官方扩展包。

当前扩展：

| 包             | 作用                                                                               |
| -------------- | ---------------------------------------------------------------------------------- |
| `@elfui/chain` | 链式编程扩展包，面向旧站渐进增强、小 demo、其它框架嵌入场景，携带 runtime compiler |

核心框架包继续放在 `packages/`。未来鸿蒙 adapter、其它框架 bridge、低代码 adapter 等可选能力也放在这里，并维护自己的文档。
